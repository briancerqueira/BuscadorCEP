# Buscador de CEP
![NPM](https://img.shields.io/npm/l/react)

# Como executar o projeto

```bash
# clonar repositório
git clone https://github.com/briancerqueira/BuscadorCEP

# instalar dependências
npm install

# executar o projeto
npm start
```
# Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- React
